Sass Color 的处理

Sass Color 的处理

技术与工具

  • map-get($palette, $color);
    这是Sass 自带的function. 我们可以用这个方式去access $palette hashmap 之中的颜色。 在Sass 里， hex颜色都直接在template 上写出，不需要用 “”， 但事实上Sass是 将这些颜色都作为string 输出的。
  • 自定义 function
    格式：
    @function [name] ([param]){ @return; }
  • @warn 关键字
    @warn 关键字在function 中使用。我们可以把它当作js 中的console.log 使用。 @warn 会在console中输出提示信息，并且不会中断function的运行。
